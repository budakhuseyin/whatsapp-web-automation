Project Title

WhatsApp Web Automatic Message Sender
This project is a Python script that sends automatic messages via WhatsApp Web using Selenium.

Features

Search and select a contact by name
Send a message multiple times
Message counter (1-message, 2-message, ...)
Input validation (empty message / invalid count)
Language-independent message sending

Technologies Used

Python
Selenium
WhatsApp Web
Chrome WebDriver

How It Works
Install dependencies:
pip install -r requirements.txt
Run the script:
python wp_bomber.py

WhatsApp Web will open in the browser
Scan the QR code with your phone
Press Enter in the terminal after WhatsApp Web is fully loaded
Enter the contact name
Enter the message
Enter how many times the message should be sent
